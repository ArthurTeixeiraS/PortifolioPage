Meus Projetos!

Uma breve descrição sobre meus projetos...

To-Do List

Uma página "web" que funciona como uma lista de tarefas, onde você pode anotar suas tarefas e marca-las como concluídas! O destaque desse projeto é o LocalStorage, que permite com que o usuário não perca suas tarefas ao fechar a guia, ou até mesmo o navegador.

Calculadora Simples

Uma calculadora, feita com o Constructor, que é um método especial para criar e inicializar um objeto através de uma Classe! Pode-se somar, subtrair, multiplicar e dividir, além da possibilidade de usar parênteses para calculos mais complexos

Analista de Números

Página web, onde o usuário digitará quantos números quiser de 1 a 100, e quando Finalizar, alguns dados como a quantidade de números digitados, o maior e menor valor, a soma dos valores e a média dos valores adicionados. Tudo isso realizado através da manipulação de Vetores com JavaScript.
